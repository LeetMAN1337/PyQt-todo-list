# PyQt-todo-list
Это приложение, в котором можно добавлять дела и время к которому их надо выполнить. Также в приложении есть кликер и полезные советы, которые можно получить по кнопке. Проект сделан на 100% на Python, также присутствует взаимодействие с базами данных  и разделение на разные программы, для разных функций. Приложение работает только в оффлайне.
